Biscuit Buddy is a comprehensive full-stack web application that bridges 
the gap between health awareness and action through intelligent AI personalization, 
real-time health tracking, emotional support, and seamless healthcare integration.

The application features a friendly mascot named "Biscuit" - a soft, rounded 
beige blob character - that guides users through personalized health journeys, 
combining physical health management with emotional wellness support.

Key Features:
• AI-powered medicine scanner using computer vision to identify tablets 
  and provide comprehensive drug information, side effects, and warnings
• Intelligent chatbot for health queries, first aid advice, medicine information, 
  and personalized exercise recommendations
• Real-time stress detection with voice note support from loved ones
• Daily streak tracking system that unlocks hospital appointment discounts
• Social features including friend challenges, leaderboards, and shared progress
• Hospital integration with nearby facility finder, online consultants, 
  and appointment booking with streak-based discounts
• Smartwatch integration for real-time heart rate, steps, and sleep monitoring
• Personalized 4-week health roadmaps based on user's health goals and conditions
• Medicine reminder system with loved one notifications for missed doses
• Comprehensive health dashboard with real-time metrics and analytics

Biscuit Buddy transforms passive health awareness into active health management 
by combining behavioral analytics, AI recommendations, social accountability, 
and healthcare system integration into one seamless companion experience.
